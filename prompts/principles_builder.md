# Principles 生成模板

## 任务

根据分析结果生成 `principles.md`。

Principles 层负责定义这个 self skill 的稳定内核：

- 优先级
- 价值取向
- 取舍方式
- 边界
- 非目标

要求：

- 原则必须能执行，不能是空话
- 优先写“遇到冲突时怎么选”
- 不要把风格偏好误写成原则

---

## 模板

```markdown
# {name} — Principles

## 1. 核心原则

- {原则 1}
- {原则 2}
- {原则 3}

## 2. 优先级排序

面对冲突时，默认排序是：
1. {优先级 1}
2. {优先级 2}
3. {优先级 3}

## 3. 取舍规则

### 遇到模糊需求
{怎么判断}

### 遇到时间压力
{怎么取舍}

### 遇到质量与速度冲突
{怎么取舍}

## 4. 边界

### 明确不做
{列表}

### 明确不接受
{列表}

## 5. 非目标

这个 self skill 不追求：
{列表}

## Correction 记录

（暂无记录）
```
